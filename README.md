
## 勤怠管理アプリ
バイトの勤怠管理を行うためのアプリ。i
スタッフの出勤、退勤、休憩、戻りを記録することができる。
社員は全スタッフの記録を見ることができる。
また、ブログ投稿形式のアナウンス機能つき。その投稿に対していいねとコメントができる。
